# Програмне забезпечення енергетичного менеджменту (ЛР №1)

[cite_start]Даний проєкт реалізує систему збору та аналізу енергоспоживання для **Готелю** (Варіант №4)[cite: 483].

## Архітектура системи
[cite_start]Система базується на трирівневій моделі обліку[cite: 44]:
1. **Рівень 1**: Головний лічильник об'єкта.
2. **Рівень 2**: Основні вузли (Харчоблок, Житловий корпус, Технічні системи).
3. **Рівень 3**: Кінцеві споживачі (Плити, Холодильники, Освітлення тощо).

## [cite_start]Технічні характеристики (Варіант 4) [cite: 483]
- **Об'єкт**: Готель (режим 24/7).
- **Площа**: 2500 м².
- [cite_start]**СЕС**: 120 кВт[cite: 240].
- [cite_start]**Батарея**: 100 кВт*год[cite: 245].

## Структура файлів
- `setup.sql`: Створення таблиць (PostgreSQL), View для рівнів обліку та ініціалізація метаданих.
- [cite_start]`generate_data.py`: Python-скрипт для генерації 8760 годин реалістичних даних (погода, споживання, СЕС)[cite: 77, 80].
- [cite_start]`queries.sql`: Набір аналітичних запитів (споживання за періодами, тарифні зони, аномалії)[cite: 91, 103].

## Встановлення та запуск
1. Виконайте `setup.sql` у вашій базі даних PostgreSQL.
2. Встановіть залежності Python: `pip install pandas numpy`.
3. Запустіть `generate_data.py` для створення CSV-файлів.
4. Імпортуйте дані в БД за допомогою команди `COPY` (шляхи до файлів вказуються в `setup.sql`).
5. Використовуйте `queries.sql` для отримання аналітики.
